# **Introduction**

This guide assumes that you have played Skyrim before and are familiar with the [Official Anniversary Edition Survival Mode](https://en.uesp.net/wiki/Skyrim:Survival_Mode). So, we'll just be sticking to just the changes that ZISS makes to the game.

Combat works exactly how you would expect from a souls like, so we won't really be spending any time talking about that either.
<br><br>
:pencil2: Checkout the [Key Combat Modifers](https://github.com/GamesRedone/ZISS/blob/main/CustomizationGuide.md#key-combat-modifiers) section of the [Customization Guide](https://github.com/GamesRedone/ZISS/blob/main/CustomizationGuide.md) to modify things such as stamina usage and damage output.

:heavy_plus_sign: With the enhanced difficulty, and changes to progression, warning labels have been added when entering new encounter zones. Learn more about these indicators in the [FAQs section](#What-is-this-new-icon-I-see-when-entering-a-new-space).

# **Difficulty Presets**

Choose how to return to Tamriel. ZISS includes six difficulty presets in the form of patches to simplify mod configuration.

👍 Just [select a Preset before you launch the game](#Selecting-a-Preset) and you're good to go.

The Default Preset is automatically loaded if no preset is selected.

- **The Default Preset** was created to add an extra challenge to the game while not making things overly difficult.
- **The Vanilla+ Preset** leaves many modifiers at their default Vanilla values.
- **The Hardcore Preset** offers a true challenge to those looking for one.

Each preset has a (No Survival) version for all the enhancements ZISS brings, minus the Survival.

## ***Selecting a Preset***

To select a difficulty preset check the box next to the preset you want to play with in Mod Organizer 2 before launching the game. The settings will be loaded automatically in game after creating a new character.

<img src="https://github.com/GamesRedone/ZISS/blob/main/Images/howto_activatepreset.jpg" loading="lazy">

The selected difficulty preset can be changed at any time with *NO ISSUES* to existing saves, just be sure to [reset your MCM Settings](https://github.com/GamesRedone/ZISS/blob/main/CustomizationGuide.md#resetting-mcm-settings-and-switching-between-difficulty-presets) when switching between presets.
<br><br><br>
:warning: The *Default (Survival) Difficulty Preset* is applied automatically with `ZISS - Settings Loader` if you do not select a Preset.

:floppy_disk: If you end up changing any of your settings, be sure to [save your changes with MCM recorder](https://github.com/GamesRedone/ZISS/blob/main/CustomizationGuide.md#saving-your-settings-with-mcm-recorder) so you don't have to go through setup again on another character.

# **Controls**

:thumbsup: You can now setup some custom keybinds on the controller!

ZISS has been configured with controller users in mind. Keyboard and mouse users will not notice much of a difference to the [controls](https://static.wikia.nocookie.net/elderscrolls/images/f/f1/Kbd-elder-scrolls-v-the-skyrim.svg/revision/latest/scale-to-width-down/1000?cb=20180421181218).
<br><br><br>
**New Additional Keyboard keybinds (Do *NOT* change):**

- `=/+` = Change shoulder Camera View
- `Shift` + `E` = [Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273) *(Allows you to pet animals among many other things)*
- `H` = Calls horse *(No matter what horse you have it will run to you)*
- `0/)` = Toggle helmet/hood on/off
- `-/_` = Toggle Photo Mode (Checkout the [FAQs section](#where-are-the-screenshots-and-save-files) to see where these are saved)

<br><br>
<img src="https://github.com/GamesRedone/ZISS/blob/main/Images/controls_controller.jpg" loading="lazy">
> *These controls can be found in game within the MCM under `"Complete Controller Setup"`*

<br>

Now a keyboard and mouse is mostly only required for text inputs, and with [Auto Input Switch](https://www.nexusmods.com/skyrimspecialedition/mods/54309) you won't have any issues switching back and forth from keyboard to controller either.

**Some additional controller keybinds, not listed above, have also been added (Do *NOT* change):**

- `A` + `D-Pad: Right` = Change shoulder Camera View
- `A` + `D-Pad: Up` = [Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273) *(Allows you to pet animals among many other things)*
- `Double Press` `D-Pad: Right` = Calls horse *(No matter what horse you have it will run to you)*

# **Alternative Start Options**

*If you would like to start the game with the Vanilla start simply select it from the statue after MCM recorder is done running.*

You now have the ability to choose from many different alternate starts to the game. Making your return to Tamriel a little more interesting by adding options to give an extra challenge or more depth to role playing.
<br><br><br>
[Why I Came to Skyrim - Origin Stories](https://www.nexusmods.com/skyrimspecialedition/mods/167166) by [jayserpa](https://www.nexusmods.com/profile/jayserpa) adds 40 possible new starting quests to give some initial purpose to your character.

[Alternate Start - Live Another Life](https://www.nexusmods.com/skyrimspecialedition/mods/272) by [Arthmoor](https://www.nexusmods.com/profile/Arthmoor) was selected as the alternate Start mod.

>*Unfortunately the mod author has removed the ability to add voice lines to the alternate start mod (There is no actor to target for the voice lines), so there are no voice lines for the alternate starts in this mod.*

<br>

:mag: You can find a full list of all the alternate start options within the mod's [Nexus description](https://www.nexusmods.com/skyrimspecialedition/mods/272).

:warning: You do not have to use one of the alternate starts, but do *NOT* disable this mod.
> *When you start in the jail cell it gives MCM recorder time to run.*

<br>

## ***Here's how the alternate start works***

No matter what alternate start you choose, you can start the main questline by talking to any innkeeper and asking about "rumors".

- When you first spawn in, after letting MCM recorder do its thing, you will be sitting in a jail cell.
- You will have the option of many different alternative starts. As well as the `Book of Origins` will be sitting on the table.
- Interact with the Statue to choose how you'll start the game.
- Interact with the Book of Origins to pick a backstory and your starting quest. If you change your mind, you can cancel it and get another one or go with no story, by interacting with the book again.
>*You can also find the book of origins inside Dragonsreach in Whiterun and in the cave in Helgen before the collapse in case you want to change this later.*
>*The new starting quests are meant to be something you look for throughout your entire playthrough, so there are no quest markers.*
>*If you are having trouble getting the Book of Origins quests to start... be sure to actually select the origin story by pressing `"A"` when using a controller.*

# **Improved AI and New Random Encounters**

Enemies and other friendly NPCs are now smarter and stronger then ever before.

:mag: Checkout the [complete overview of the difficulty presets](#Difficulty-Presets) to learn more about their differences.

- You can expect to find yourself in quiet a few new situations with over 300 new random encounters
- MINPCs adds a new party system allowing you to team up with many new generic and unique NPCs
- Guards and other friendly NPCs will check your bags for stolen goods, get suspicious when your sneaking, and call you out on necromancy or when you go invisible
- NPCs now have a chance to use potions *(disabled with the Vanilla+ difficulty preset)*
- Hundreds of NPCs have new voice lines and dialog options have been added

### :snowflake: **Reworked Survival Mechanics**

*Most of these features are disabled in the [(No Survival) presets](https://github.com/GamesRedone/ZISS/blob/main/StarterGuide.md#difficulty-presets)*

In the Vanilla Survival Mode needs can almost be entirely ignored and don't prove to be that much of a challenge. While playing ZISS you will have to keep track of them a little more.

- Most menus other then the system menu (pause menu) will not pause the game. No more going through your bag right in front of the enemy. *([Can be Disabled, new save required](https://github.com/GamesRedone/ZISS/blob/main/CustomizationGuide.md#optional-mods))*
- There is far less food in the world and you must eat at least 3 times a day to avoid debuffs.
- You need to sleep once a day and the quality of your sleep is determined by what you are sleeping in. *(Beds will give you the best sleep, hay piles the worst)*
- Your cold exposure rate is now based not only on your location but, your altitude, the weather severity, and your clothing warmth. The bar under your needs icons represent your exposure. The small bar below that represents how wet your are.
- Food has varying levels of nutrition with larger items being able to satisfy your hunger more and your food is now able to spoil. Spoilage is indicated as a (%) next to the food item in your inventory.
- Like Vanilla survival mode, fast travel is disabled, but the ferry and carriage system has been expanded, you can now wait for carriages at inns, and you can craft a travel pack to fast travel from anywhere
- There are now a variety of different types of coins other then Septims around the world depending on the region your in. They are automatically exchanged for their gold value, but they all have varying exchange rates. *(All coin values are set to 1 Septim in the Vanilla+ Preset)*
- You can crank up the "Realism" even more and require water to be purified or make diseases deadly with many options in the MCM under iNeed

  > *One in game day is roughly 1 hour, meaning you must tend to most needs roughly 2-3 an hour and sleep once roughly every hour.*

### :skull: **Alternative Death System**

*Disabled completely in the [Vanilla+ and (No Survival) presets](https://github.com/GamesRedone/ZISS/blob/main/StarterGuide.md#difficulty-presets)*

ZISS uses [Shadow of Skyrim - Nemesis and Alternative Death System](https://www.nexusmods.com/skyrimspecialedition/mods/65136), a highly configurable alternative death mod to provide a continuous gameplay experience without saving and reloading.

- With the default difficulty preset, if you die you will end up waking up in one of the last locations you visited, 6 in game hours later, injured and missing 25% of your gold
- With the hardcore difficulty preset you'll wake up in a random location with a chance to even spawn in a hostile location. You will also, have to hunt down who killed you to remove a small debuff. When you kill them you will receive a small buff. With a max of 5 debuffs or buffs.
- You can retrieve your missing gold by locating your dropped backpack at your place of death. It will remain there even if you die a second time and a torch will be lit near your backpack to show its location. 

  > *Dropped backpacks and NPCs you need to hunt down can be found in the MCM under Shadow of Skyrim*<br><br>*If you cannot recover a backpack go to Shadow of Skyrim in the MCM > go to the `Developer Mode` tab > click `Recover Lost Items`.*

## ***Open World Loot***

Over the course of over a year [deletepch](https://www.nexusmods.com/skyrimspecialedition/users/760981) has modified each individual dungeon and Encounter Zone hand setting the min and max level of enemies. The game will not be overly difficult at first, and difficulty will ramp up as you continue through the story.

> ***Encounter Zones*** 
> *are hidden, boundary-defined areas (like dungeons or specific regions) that control the level scaling of enemies and loot. Encounter Zone's spawn enemies and loot relative to your character's level when you first enter*

Rather then deleveling the world, leading to you being trapped, without the ability to go anywhere you want in the beginning of the game... OWL's modifications to Skyrim give you the freedom to explore Skyrim at your own pace and desire while still giving you the feeling of true progression.

Dungeons and Encounter Zones containing powerful enemies, unique bosses or artifacts have higher min/max levels. With the best gear being restricted to the boss chests of the high level enemies.

<br>

**The creator of OWL describes progression in 4 phases:**

- **Level 1 to 15** is considered to be Early Game, you mostly deal with humanoid enemy and doing some early stage main quests and guild quests

- **Level 15 to 30** is regarded as Mid Game, at this stage you should have gathered some solid gear and have a sufficient build for mid stage main quests and finish the guild quests

- **After level 30** you have entered the Late Game. You should have a concrete build for most challenging part of the game. You started to hunt down the appropriate artifacts for your build.

- **At around level 40 to 50**, you have reached the END GAME

## ***Experience and Leveling Freedom***

[Experience](https://www.nexusmods.com/skyrimspecialedition/mods/17751) and [Leveling Freedom](https://www.nexusmods.com/skyrimspecialedition/mods/69589) work together to give you control over how XP is earned and scales.

<br>

In order to provide smoother progression mid to late game and keep things from feeling completely handed to you in the start. Leveling Freedom increases the initial XP requirements for each level, and decreases from the Vanilla values as the game goes on.

*The chart below shows the changes to XP scaling.*

<img src="https://github.com/GamesRedone/ZISS/blob/main/Images/xpscaling.png" loading="lazy">

<br>

Experience restricts XP gain solely to quests and exploration. 

> *Skills no longer affect your character level and skills are now capped relative to the character level.* 

***What this means is...***

- Leveling is separate from increasing your individual skills.

- You won't have to grind as much.

- You no longer can level up your overall character by repeatedly casting Muffle to raise Illusion or crafting hundreds of iron daggers to raise Smithing.

- The world will not become overleveled because of high level non-combat skills.
    
    > *This is key to the changes [OWL](#Open-World-Loot) makes to Encounter Zones*
  
<br>

**The Skill Cap**
> *Disabled completely in both the [Vanilla+ and Vanilla+ (No Survival) presets](https://github.com/GamesRedone/ZISS/blob/main/StarterGuide.md#difficulty-presets)*

Skill cap is calculated with the following equation:

`fSkillCapBase + (playerLevel * fSkillCapMult) = skillCap`

Across all difficulty presets the skill cap has been set to:

`18 + (playerLevel * 2) = skillCap`

What this means is...

**At Level 1 your skill cap will be 20**
    
    18 + (1 * 2) = 20

**At Level 5 your skill cap will be 28**

    18 + (5 * 2) = 28

## ***Why is there no quicksave***

*Don't worry the game still autosaves.* Your game will save every 15 minutes automatically and when your character sleeps.

The vanilla quicksave & autosave features do not properly halt active scripts before taking a snapshot of your game. This can cause your game to crash and can even cause save files to become corrupted among many other issues. Because of this, the feature has been disabled altogether.

<br>

:warning: It is highly recommended you do *NOT* disable any of the mods that disable the quicksave feature. 

:pencil2: You can configure how often your game saves in the MCM under [SafeSave System Overhaul](https://github.com/GamesRedone/ZISS/edit/main/CustomizationGuide.md#key-balancing-changes).

>*I would not recommend decreasing the time between saves too much. Since all saves with SafeSave System Overhaul are hard-saves, you may expereince a slight stutter when the game saves. SafeSave System Overhaul tries to do this when you are not moving or are in a safe place, but if the time between saves is too short you may start to notice the stutter.*

<br>
