## Reporting a Vulnerability

If you find a security vulnerability, copyright issue, or a maliciously flagged file in this Wabbajack modlist, please do not report it publicly as a GitHub issue. Instead, please contact the maintainers directly via:

* **Nexus :** [GamesRedone](https://www.nexusmods.com/profile/GamesRedone)

## Responsible Disclosure

Please allow the maintainers reasonable time (typically 7-14 days) to investigate and patch any reported issues before publicly disclosing them. 

## Supported Versions

Security updates and critical fixes are provided on a best-effort basis for the following versions of this modlist:


| Version | Supported |
| :--- | :--- |
| v1.x (Latest) | Yes |
